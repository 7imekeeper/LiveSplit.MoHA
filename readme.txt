This is a LiveSplit (livesplit.org) component for Medal of Honor: Airborne. It removes load times
automatically and automates splitting.

Features:

	Frame-accurate load time removal.

	Automatic reset, start, and split. Automatic splitting can be done on
	mission end screens as well as at a few other common places.

Requirements:

	Medal of Honor (PC)
	LiveSplit 1.6+

Install:

	Enter "Medal of Honor: Airborne" in LiveSplit's splits editor. Click activate.
